# CAudio Samples

`CAudioSampleController` demonstrates the main runtime APIs in a scene-friendly form.

Usage:

1. Create or open a scene.
2. Add an empty GameObject.
3. Attach `CAudioSampleController`.
4. Assign an `AudioDatabase`, optional `followTarget`, and optional `directClip`.
5. Call the public methods from UI Buttons, animation events, or editor debug tooling.

Covered APIs:

- SFX/UI/Voice/Ambience/Music playback.
- Music crossfade.
- 3D position playback.
- Follow-target playback.
- Direct clip playback.
- Loop handle stop.
